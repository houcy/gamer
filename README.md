# Gamer AI Laboratory

An AI Laboratory for JavaScript, designed to help students (A) acquire programming practice, and (B) develop AIs based on MiniMax.

Gamer simplifies development for games like:

- Tic Tac Toe
- Connect Four
- Othello
- Checkers
- Chess
